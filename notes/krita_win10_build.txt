Steps for building Krita in win10

https://docs.krita.org/en/untranslatable_pages/building_krita.html#building-on-windows

Requirements:
- Git
- CMake > 3.10
- mingw-w64 7.3
- Python 3.8
- Window 10 SDK
- QT Creator

Steps:
1. prepare dir
- krita_dev
  - dev
    - Krita (git source)
  - deps
  - install
2. create krita/env.bat
